# Project Todo

Manage a persistent todo list stored in `.claude/TODO.md`.

## Commands:
- `/todo` - Show all todos
- `/todo add <task>` - Add a new task
- `/todo done <number>` - Mark task as done
- `/todo remove <number>` - Remove a task
- `/todo clear` - Clear completed tasks

## Format in TODO.md:
```markdown
# Project Todo List

## In Progress
- [ ] Task description

## Completed
- [x] Completed task
```

Keep the list organized and update timestamps.
