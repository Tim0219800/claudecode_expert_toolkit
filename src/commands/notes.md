# Project Notes

Manage quick notes stored in `.claude/NOTES.md`.

## Commands:
- `/notes` - Show all notes
- `/notes add <note>` - Add a new note with timestamp
- `/notes search <query>` - Search in notes
- `/notes clear` - Clear all notes

## Format:
```markdown
# Project Notes

## 2025-01-15 14:30
Note content here...

## 2025-01-15 10:15
Earlier note...
```

Notes are useful for:
- Architectural decisions
- Meeting notes
- Ideas to explore later
- Debugging observations
